# linux-terminal-lab

In this repository there are script that have different purposes. There is also a recap file and a README file to detail what's expected from each folder and file. The recap file recaps each section and answers debugging scenarios. To run any of the scripts, go into the script folder, take a script you would like to run and go to command prompt or any shell and input a command like this without the quotation marks to run the script "./scriptname.sh."
